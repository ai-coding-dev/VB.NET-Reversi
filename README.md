A simple and lightweight Reversi (Othello) game built using **VB.NET** and **Windows Forms**. It supports local two-player gameplay with automatic move validation and piece flipping logic.

### Features
* **Automatic Rules**: Validates moves based on classic Reversi rules.
* **Score Tracking**: Real-time display of Black and White piece counts.
* **Game Loop**: Detects when a player has no moves and automatically passes the turn or ends the game.
* **Modern .NET**: Compatible with .NET 8.0.

### How to Run
1.  Ensure you have **Visual Studio 2022** and the **.NET 8 SDK** installed.
2.  Open `VB.NET-Reversi-main.sln`.
3.  Press `F5` to build and run the application.
