<a name="english"></a>
## 🇺🇸 English
A simple and lightweight Reversi (Othello) game built using **VB.NET** and **Windows Forms**. It supports local two-player gameplay with automatic move validation and piece flipping logic.

### Features
* **Automatic Rules**: Validates moves based on classic Reversi rules.
* **Score Tracking**: Real-time display of Black and White piece counts.
* **Game Loop**: Detects when a player has no moves and automatically passes the turn or ends the game.
* **Modern .NET**: Compatible with .NET 8.0.

### How to Run
1.  Ensure you have **Visual Studio 2022** and the **.NET 8 SDK** installed.
2.  Open `VB.NET-Reversi-main.sln`.
3.  Press `F5` to build and run the application.

<a name="chinese"></a>
## 🇨🇳 中文
这是一个使用 **VB.NET** 和 **Windows Forms** 开发的黑白棋（奥赛罗）经典游戏。支持本地双人对战，具备自动规则校验和翻子逻辑。

### 功能特点
* **规则自动化**：基于经典黑白棋规则自动校验合法落子位置。
* **实时计分**：顶部标题栏实时显示黑白双方的棋子数量。
* **自动判定**：当一方无棋可走时自动跳过，并能判定游戏结束与胜负。
* **现代框架**：基于 .NET 8.0 开发，运行高效。

### 如何运行
1.  确保已安装 **Visual Studio 2022** 和 **.NET 8 SDK**。
2.  打开 `VB.NET-Reversi-main.sln` 解决方案文件。
3.  按 `F5` 键编译并启动程序。

<a name="japanese"></a>
## 🇯🇵 日本語
**VB.NET** と **Windows Forms** を使用して作成された、シンプルなリバーシ（オセロ）ゲームです。ローカルでの2人対戦に対応しており、合法手のチェックや石を裏返すロジックが実装されています。

### 主な機能
* **ルール自動判定**: クラシックなリバーシールールに基づき、石を置ける場所を自動で判定します。
* **スコア表示**: 黒と白の石の数をリアルタイムでタイトルバーに表示します。
* **パス・終了判定**: 置ける場所がない場合のパス処理や、対局終了時の勝敗判定を自動で行います。
* **最新環境**: .NET 8.0 に対応しています。

### 実行方法
1.  **Visual Studio 2022** および **.NET 8 SDK** がインストールされていることを確認してください。
2.  `VB.NET-Reversi-main.sln` を開きます。
3.  `F5` キーを押して、ビルドおよび実行を行います。
